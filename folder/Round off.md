# Round off to nearest


```python
#round off till one decimal point

print(round(5.265, 1))
print(round(5.276, 1)) 
print(round(5.273, 1)) 
```

    5.3
    5.3
    5.3
    


```python
#round off till two decimal point

print(round(5.265, 2))
print(round(5.276, 2)) 
print(round(5.273, 2))  
```

    5.26
    5.28
    5.27
